## 大学生线上学习资源共享与问答系统`

**描述**：为提高大学生在线学习效率，搭建一个学习资源共享与答疑互动平台支持学生上传学习资料、提出学习问题、教师进行答疑、管理员进行平台维护。

(1)团队成员(3-5人一组)
(2)开放github站点分享，编写readme文档(markdown)，要包含成员组成、分工、基本设计思路、技术选型和开发进度安排。

### 功能

#### 1. 系统角色说明
系统包含三类用户：
- 管理员账号：系统初始化创建；
- 学生账号：可自由注册；
- 教师账号：由管理员添加


#### 2. 管理员功能模块

* **课程管理**：增删改课程信息，包括课程名称，授课教师，课程内容描述，开课学院。
* **教师管理**：增删改教师信息;设置教师讲授课程(可多个)、简介、职称。
* **学习资源管理**：删除不合规或违规的资源与资料;修改资源说明信息。
* **问答内容管理**：删除、修改任意学生提问、教师回答;管控错误或违规内容。


#### 3. 学生功能模块
描述：学生登录后可参与资源共享与学习问答。

* **通知提醒**：若发布的问题被教师回答，系统提示：“您有 X条新回答”，可点击查看
* **学习资源浏览**：按课程分类浏览所有学习资源;支持关键字搜索(标题/内容)。
* **上传学习资料**：学生可上传自己的学习资源(如 PDF、图片、压缩包)，并填写：资源标题、简介、所属课程、上传时间(系统记录)。
* **查看资源详情**：上传者;资源简介;下载次数。
* **学生问答区**，学生可进入某个课程页面提出问题：提问标题、内容、时间;支持上传图片作为附件;支持分页显示问题列表。
* **提问搜索**：按学科、教师、关键字检索全站问答,
* **学生个人中心**：查、改、删自己上传的学习资料;查、改、删自己提出的问题;查看已被回答的问题及回答内容。

#### 4. 教师功能模块
描述：教师登录后主要负责答疑和资源维护。

* **新问题提醒**，显示：“您所上课程有 X条未回答问题”可点击进入答疑页面。
* **发布学习资源**，教师可上传高质量资料，填写：资源标题、说明内、所属课程、可上传附件(PDF、图片等)。
* **回答学生问题**，教师可：查看所上课程的全部学生提问、回答问题(支持文本 + 图片附件)、修改或删除自己的回答。
* **学科权限设置**，教师可设置：某课程学习资源仅对本班学生可见或全部学生可见。5、通用功能模块
  所有用户均可：修改个人密码，修改个人资料(头像、邮箱等)
